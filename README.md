# 📊 ZipRide Financial Dashboard

## Overview
A financial performance dashboard built in Excel tracking revenue, 
operational costs, and profitability across 3 cities (Bengaluru, Mumbai, 
Hyderabad) over H1 2025 for a fictional bike rental company.

## Key Features
- **Power Query** data pipeline from CSV for auto-refreshable data
- **SUMIFS** formulas for dynamic P&L aggregation across cities and months
- **INDEX-MATCH** for category lookups across sheets
- **Variance analysis** auto-flagging months where revenue misses target
- **3 charts** — Revenue vs OpEx, City Revenue Split, Gross Margin Trend
- **Conditional formatting** for instant visual anomaly detection

## Business Questions This Dashboard Answers
1. Which city generates the highest revenue?
2. Are we hitting monthly revenue targets?
3. Is gross margin improving or declining month over month?
4. Which months need management attention?

## Key Insights from the Data
- Bengaluru contributes 38% of total revenue — highest among 3 cities
- Gross margin peaked in March 2025 and declined through June
- Revenue consistently exceeds OpEx across all 6 months
